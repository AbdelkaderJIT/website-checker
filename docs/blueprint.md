# **App Name**: ClarityGrader

## Core Features:

- URL Input: Accept a website URL from the user for analysis.
- Website Parsing: Automatically crawl and parse the website content to extract text, structure, and image metadata.
- Plain Language Analysis: Analyze the extracted content against plain language standards to assess text clarity, structural clarity, and visual clarity.
- Score Calculator: Use an AI tool to calculate scores in the different grading categories, following grading rubrics.
- Report Generation: Generate a detailed plain language report with an overall score, category breakdowns, highlighted issues, and specific recommendations for improvement.
- Report Storage: Store the generated reports in Firestore for each analyzed domain.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey intelligence and trustworthiness.
- Background color: Light lavender (#EDE7F6), a desaturated variant of the primary, for a gentle and professional feel.
- Accent color: Electric blue (#3D5AFE), an analogous hue with higher brightness and saturation, to highlight key actions and data.
- Headline font: 'Space Grotesk' sans-serif for a techy, scientific feel; use 'Inter' for body text
- Note: currently only Google Fonts are supported.
- Use clear and intuitive icons to represent different categories of analysis and report sections.
- Single-page layout with a clean and structured design for easy navigation and understanding of the report.
- Subtle loading animations and transitions to provide a smooth user experience while the website is being analyzed.
